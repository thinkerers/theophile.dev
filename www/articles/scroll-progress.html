Keep scrolling !

This technique employs a :after pseudo-element. A rectangle is drawn in the background with linear-gradient. It is then hidden using a clip path. As the user scrolls, the clip-path stays in place with background-attachment: fixed, and the hidden rectangle gradually comes into view.

A drawback of this approach is a slant at the end of the indicator bar, caused by the clip path shape. It is also not easy to control the speed of the "animation". This will be especially noticeable if the content to be scrolled is short. Using animation timeline would provide a better, smoother solution, but as of now (february 2024), widespread support for this feature is lacking.
